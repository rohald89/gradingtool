Treehouse Grading Tool

This is an application designed to make grading a student's work easier, and also giving the student a very well structured, informative review. As this application contains Treehouse intellectual property, please do not share this tool outside of the Treehouse Moderators channel within Slack.


To get started, choose a Techdegree from the list in the dropdown in the top-left of the grading tool. You can favorite a Techdegree by clicking the next to the Techdegree name. This will load that Techdegree by default when you first open the grading tool. To get started grading a project, select a project from the left panel. Once one is selected you'll be able to review the requirements for that project and start grading.
